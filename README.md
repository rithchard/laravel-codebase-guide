## üìò Description

### **Migration1.php**
Defines the creation of a new table with proper data types, constraints, and timestamps.  
Includes comments explaining each column and migration method.

### **Migration2.php**
Demonstrates how to alter or extend an existing table, showing how to manage relationships or add foreign keys.

### **Model.php**
Shows a fully documented Eloquent model.  
Includes examples of:
- `$fillable` and `$hidden` attributes  
- Relationships (`hasOne`, `belongsTo`, etc.)  
- Common query scopes

### **Controller.php**
Contains a basic CRUD controller.  
Each method (`index`, `store`, `update`, `destroy`) is clearly documented to show its interaction with the model and database layer.
